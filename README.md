# Go File Tracker

## Описание
Приложение для отслеживания изменений в файлах с использованием языка Go. Позволяет мониторить изменения и вести лог изменений.

## Технологии
- Go

## Установка и запуск
```bash
git clone https://github.com/Handruka/go-file-tracker.git
cd go-file-tracker
go run main.go
